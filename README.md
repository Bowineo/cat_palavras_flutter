Based on the successful game Torto from Coquetel magazine, this is a catified version! Form words going in all directions, always connecting the letters in direct sequence, without skipping and without repeating letters (for a word to have a repeated letter, it is necessary that the same letter is also duplicated in the diagram). Only words of 4 letters or more are valid. Good luck and have fun!
